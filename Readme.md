Nombre del proyecto:
Luking Store

Tecnologias usadas:
Python
Git
Github
Entorno virtual (venv)
Visual Studio

Problematica que resuelve:
Nuestro proyecto busca solucionar la falta de organización en el registro de productos y ventas dentro de una tienda de ropa. Muchas veces estos datos se llevan de forma manual, lo que puede generar errores o pérdida de información. El sistema permite registrar productos como gorras, zapatos, jeans y camisas, además de llevar un control básico de las ventas y del inventario, facilitando la administración de la tienda.

Como poner el codigo en marcha:
Clonar el repositorio: https://github.com/danielG-blip/entrega_1.git,
Se debe crear un entorno virtual para instalar las librerias de panda y el requirements.txt.
